== Description ==

This GamiPress add-on adds a global switch in the Backend where the awarding of badges can be enabled and disabled.

**Note:** You will need to install the free [GamiPress plugin](https://wordpress.org/plugins/gamipress/ "GamiPress")&trade; (version 1.4 or higher) to use the GamiPress Activation Add-on. The Plugin was tested up to GamiPress 1.9.4

[Get the GamiPress plugin](http://wordpress.org/extend/plugins/gamipress/ "GamiPress").

This plugin was originally developed for the [Academy of International Cooperation](https://www.giz.de/akademie/en/html/index.html) of [GIZ](https://www.giz.de).

== Installation ==

1. Upload, activate and configure the free [GamiPress plugin](https://wordpress.org/plugins/gamipress/ "GamiPress") to WordPress.
2. Upload 'activation-add-on-for-gamipress' to the '/wp-content/plugins/' directory
3. Activate the plugin through the 'Plugins' menu in WordPress

== Screenshots ==

1. Activation Switch, if turned off, it overrides the course settings. If turned on, badges can be earned during the set course dates.
